# ArgoCD GitOps with K3d

### Install kubectl and k3d on local machine

on Mac using Homebrew

```bash
brew install kubectl k3d
```

### Commands

**Start Local ArgoCD**

```bash
make start-local-argo
```

**Stop Local ArgoCD**

```bash
make stop-local-argocd
```
